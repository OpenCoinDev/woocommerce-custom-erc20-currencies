## 安装

1. 从 [WordPress 官方插件市场](https://wordpress.org/plugins/woo-custom-erc20/) 下载并安装
2. 从 Github 上下载并安装

## 使用

1. 安装插件
2. 激活插件
3. 在「设置」- 「WooCommerce ERC20 货币自定义」 中设置你的 ERC20 Token 并保存
